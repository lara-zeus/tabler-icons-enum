## Introduction

@zeus tabler-icons-enum provides Tabler icons as enums for easy usage with Filament.

**[Github](https://github.com/lara-zeus/tabler-icons-enum) · [Packagist](https://packagist.org/packages/lara-zeus/tabler-icons-enum) · [Discord](https://discord.com/channels/883083792112300104)**

## Features

- Strongly-typed icon references via enums
- Works directly with Filament icon properties
- Lightweight package for UI consistency
